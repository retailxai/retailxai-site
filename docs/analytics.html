Analytics Setup

Privacy-friendly analytics integration

Plausible Analytics

RetailXAI uses Plausible Analytics, a privacy-friendly, GDPR-compliant analytics solution that doesn't use cookies or collect personal data.

Setup Instructions

  1. Sign up for a Plausible account at plausible.io
  2. Add your domain (retailxai.github.io) to your Plausible account
  3. The analytics script is already included in index.html:
<script defer data-domain="retailxai.github.io" src="https://plausible.io/js/script.js"></script>

Custom Event Tracking

Custom events are tracked using the data-analytics attribute:

<a href="..." class="btn" data-analytics="cta-hero-get-started">Get Started</a>

The tracking script automatically captures clicks on elements with the data-analytics attribute.

Tracked Events

  • cta-hero-get-started - Hero "Get Started" button
  • cta-hero-why - Hero "Why RetailXAI" button
  • footer-github - Footer GitHub link
  • footer-substack - Footer Substack link
  • footer-producthunt - Footer Product Hunt link

Alternative: GoatCounter

If you prefer GoatCounter, replace the Plausible script with:

<script data-goatcounter="https://YOUR-CODE.goatcounter.com/count" src="https://gc.zgo.at/count.js"></script>

Privacy & Compliance

Both Plausible and GoatCounter are:

  • GDPR compliant
  • Cookie-free
  • No personal data collection
  • Lightweight and fast
