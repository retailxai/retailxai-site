Getting Started with RetailXAI

Welcome to RetailXAI! This guide will help you get up and running quickly with our content intelligence pipeline.

What is RetailXAI?

RetailXAI is an AI-powered platform that transforms video content into professional articles. It can ingest YouTube videos, earnings calls, and podcasts to produce publication-ready content for Substack, LinkedIn, and X.

Prerequisites

Before you begin, ensure you have:

  • Python 3.10+ installed
  • Git for version control
  • YouTube API access (optional, for enhanced metadata)
  • OpenAI/Anthropic API keys (optional, for AI features)

Quick Installation

1. Clone the Repository

git clone https://github.com/retailxai/precipice-2.git
cd precipice-2

2. Set Up Python Environment

# Create virtual environment
python3 -m venv .venv

# Activate virtual environment
# On macOS/Linux:
source .venv/bin/activate
# On Windows:
.venv\Scripts\activate

# Install dependencies
pip install -e ".[dev]"

3. Configure Environment

# Copy environment template
cp env.example .env

# Edit with your settings
nano .env  # or your preferred editor

Required Environment Variables:

# Database
DATABASE_URL=sqlite:///./precipice.db

# AI Services (at least one required)
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key

# YouTube API (optional)
YOUTUBE_API_KEY=your_youtube_key

Your First Pipeline Run

Basic Usage

# Process a YouTube video to article
precipice process "https://www.youtube.com/watch?v=VIDEO_ID"

Expected Output

The pipeline will:

  1. Ingest video metadata and transcript
  2. Analyze content for key points
  3. Generate a professional article
  4. Save the output to your data directory
