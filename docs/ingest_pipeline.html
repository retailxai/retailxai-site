Ingest Pipeline

The RetailXAI ingest pipeline processes content from multiple sources and transforms it into professional articles.

Pipeline Stages

1. Source Ingestion

Ingest content from various sources:

  • YouTube Videos: Extract metadata and transcripts
  • RSS Feeds: Process podcast episodes
  • Direct Audio Uploads: Support for local files
  • Article URLs: Extract and process web content

2. Transcription

Convert audio to text using multiple services with intelligent fallback:

  • Local Whisper: Fast, offline transcription
  • OpenAI Whisper API: High-quality transcription
  • Google Speech-to-Text: Backup service

The system automatically selects the best available service and falls back if errors occur.

3. Content Analysis

Analyze transcribed content to extract key information:

  • Auto Tagging: Identify topics and themes
  • Classification: Determine content type (earnings call, interview, etc.)
  • Summary Extraction: Extract key points and highlights
  • Title Rewriting: Optimize titles for engagement

4. Article Generation

Generate professional articles using AI models:

  • Multiple Styles: Analytical, narrative, data-driven, opinion
  • Model Selection: Choose between OpenAI GPT and Anthropic Claude
  • Quality Control: Automatic fact-checking and quality metrics

5. Publishing

Distribute articles to publishing platforms:

  • Substack: Direct publishing integration
  • LinkedIn: Professional network sharing
  • X (Twitter): Social media distribution
  • Markdown Export: For manual publishing

Multi-Source Processing

RetailXAI can combine multiple sources into a single comprehensive article:

  • Combine multiple articles on the same topic
  • Merge podcast episodes with article content
  • Create sentiment echo analysis across sources

Error Handling

The pipeline includes robust error handling:

  • Automatic retries with exponential backoff
  • Circuit breakers to prevent cascade failures
  • Graceful degradation when services are unavailable
  • Comprehensive logging for debugging

Performance

Typical processing times:

  • Short videos (5-10 min): 2-3 minutes
  • Medium videos (10-30 min): 5-8 minutes
  • Long videos (30+ min): 10-15 minutes
  • Multi-source articles: 15-20 minutes
