## Purpose

PDF files are generated by Precipice-2 and copied to this directory by the Update Data workflow. These PDFs are associated with article drafts and can be previewed in the Svelte viewer.

## Usage

PDFs are automatically copied from Precipice-2 during scheduled workflow runs. They are deployed as part of the static site and accessible via the viewer interface.
